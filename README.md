# Inventory Tracker

## Project Overview

The Inventory Tracker is a web-based inventory management application developed using HTML, CSS, and JavaScript. The system helps businesses and users efficiently manage products, monitor stock levels, generate inventory reports, and organize inventory records through a user-friendly dashboard.

The application provides a centralized platform for tracking inventory information and maintaining accurate stock records.

---

## Objectives

The primary objectives of this project are:

* Manage inventory efficiently.
* Track product stock levels.
* Maintain product records.
* Generate inventory reports.
* Monitor inventory performance.
* Improve inventory organization and control.

## How to Run the Project

### Using VS Code

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open index.html.
4. Right-click and select "Open with Live Server".
5. The Inventory Tracker application will launch in your browser.

### Using a Browser

1. Open the project folder.
2. Double-click index.html.
3. The Inventory Tracker application will start.
